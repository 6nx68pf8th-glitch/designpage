# designpage
This is a simple Svelte project that currently consists of two pages: `About` and `Contact`. The app is built with SvelteKit, a framework for building fast, resilient web apps.

## Installation
To see it for yourself, make sure you have Node.js installed on your machine. Then follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/username/my-svelte-project.git
cd my-svelte-project
```
2. Install dependencies with npm:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```
4. Open your browser and visit the link provided to view your Svelte app.
